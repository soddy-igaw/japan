## 교통

### 나리타 → 호텔 (가는 날)

| 방법 | 경로 | 소요 | 비용 | 비고 |
|------|------|------|------|------|
| **⭐ 리무진버스 → T-CAT (추천)** | 나리타 → T-CAT(하코자키) → 택시 10분 or 도보 20분 → 호텔 | **약 55분** + α | **3,100엔** + 택시 ~1,000엔 | 짐 있을 때 환승 없이 가장 편함! |
| 스카이라이너 + 전철 | 나리타 → 닛포리 → 아키하바라 → 카야바초 → 도요초 | ~1시간 10분 | ~2,800엔 | 가장 빠르지만 환승 2~3번. 짐 들고 계단 |
| NEX + 전철 | 나리타 → 도쿄역 → 토자이선 → 도요초 | ~1시간 30분 | ~3,500엔 | 환승 1번이지만 비쌈 |

- 에비스역 앞 (숙소에서 ~25분). 야키토리·가츠오타타키·모츠나베 등 10곳+
